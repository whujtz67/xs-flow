# Function Spec Templates

This document defines reusable specification templates for describing Functions inside a TECH Specification Set.

It serves the Architect AI when documenting Functions that appear within a specific Module, Bundle, Object, or Trait. Each code structure may contain its own Functions section, where Functions should be ordered according to the code organization order or user-specified order within that structure, rather than grouped by a global Function category.

When the Architect AI encounters a Function, it should first identify the Function category, then select the corresponding template from this document and fill all required items, as well as any contextually necessary optional items, to produce a normalized specification of that Function.
